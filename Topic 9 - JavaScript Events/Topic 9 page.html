JavaScript Events Especially for Mobile Devices (onTouchBegin) and Animation and Transition Events

Topic 9 is all about JavaScript events. In the system there is a recorded of what events happen. Some of these events might be that you hovered over a button or you scrolled down the page. You may have pushed a button on your keyboard. These events are tracked and many more. The great part about them is that we can connect other actions to these events.

On all of the other pages you have seen buttons, that when clicked produce something. This is a simple 'onclick' event that runs a function when ever you click the button.
