Object Creation Functions, Properties, Methods, Instantiation

Here is my teaching session: Objects

Topic 2 focused on object creation and their accompaning properties and methods. Objects have properties which you will see below. an objects properties can also be functions.

Part 1 - Here are two ways to write a object creation function. This is what is know as object literal notation. The properties are made up of key:value pairs. 

var car1 = {make:"Ford", model:"Mustang", firstYear:"1964"};
var car2 = {
    make:"Dodge", 
    model:"Charger", 
    firstYear:"1966",
    fullName: function() {return this.make + " " + this.model}
};

Part 2 - This is an example of a constructor function. This provides a blue print for the Car object.

function Car(make, model, year, color) {
 		this.make = make;
  		this.model = model;
  		this.year = year;
  		this.color = color;
}

Part 3 - We would then be able to create an instance of the object by creating a variable and passing in the known attributes.

var mycar = new Car('Eagle', 'Talon TSi', 1993);
