JSON Parsing & Stringify

Topic 3 was all about parsing and stringifying data. Both are JSON methods that are used to convert data. They are opposites of each other.

The stringify method is first. Stringify is used when sending data to a web server. The data has to been a string. You can also use stringify if you would like to display a string value in the browser.
