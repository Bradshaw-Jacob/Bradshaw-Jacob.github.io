CSS3 Animations

Here are some animations using CSS. You can use these types of animations without using any Javascript. You most likely saw how the title of this page moved back and forth once the page was loaded.

Here is my teaching session: CSS3 Animations

The animations are made up of 2 main components. The CSS definition and the keyframes. In the definition you add many different properties. Some of these properties control the duration of the animation, or the delay before the animation starts. There are many properties that you can use.

Here is what the title animation looks like.

#title {
	animation-name: moveTitle;
	overflow: hidden;
	animation-duration: 3s;
	animation-fill-mode: forwards;
	animation-delay: 1s;
}

@keyframes moveTitle {
	0% {margin-left: 0%;}
	50% {margin-left: 65%;}
	100% {margin-left: 0%;}
}
