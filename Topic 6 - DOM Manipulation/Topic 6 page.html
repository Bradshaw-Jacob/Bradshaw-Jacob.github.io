DOM Manipulation Using createElement, appendChild, insertBefore

DOM stands for Document Object Model. The DOM ia an API that is a model of the HTML or XML document. Using the DOM you can interact with the document. You can do things such as replace text, create new HTML elements, and modify CSS properties. The DOM is one of the most popular and used APIs on the web.
